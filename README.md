# 局域网五子棋

一个零依赖的局域网五子棋小游戏。启动后，同一局域网内的玩家访问主机地址即可进入大厅，看到在线玩家并邀请对战。

## 启动

```bash
npm start
```

服务默认监听 `5173` 端口。启动后终端会显示类似：

```text
LAN Gomoku is running at http://localhost:5173
LAN addresses:
  http://192.168.1.23:5173
```

把局域网地址发给其他玩家，大家用浏览器打开即可。

## 功能

- 在线玩家列表
- 玩家邀请与接受/拒绝
- 双人五子棋对局
- 胜负判定、平局判定、认输
- 对手离线自动结束对局
