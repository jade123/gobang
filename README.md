# 局域网五子棋

一个零依赖的局域网五子棋小游戏。启动后，同一局域网内的玩家访问主机地址即可进入大厅，看到在线玩家并邀请对战。

## 启动

进入项目目录：

```bash
cd /Users/jade/Documents/局域网五子棋
```

启动服务：

```bash
npm start
```

服务默认监听 `5173` 端口。启动后终端会显示类似：

```text
LAN Gomoku is running at http://localhost:5173
LAN addresses:
  http://192.168.1.23:5173
```

把局域网地址发给其他玩家，大家用浏览器打开即可。

进入大厅后，点击 `AI模式` 可以直接开始单人对局。玩家执黑先手，AI 执白并在玩家落子后自动回合，直到分出胜负或平局。

## 访问

本机访问：

```text
http://localhost:5173
```

局域网内其他设备访问：

```text
http://192.168.1.23:5173
```

这里的 `192.168.1.23` 只是示例，请以启动终端里实际显示的 `LAN addresses` 为准。

## 关闭

如果服务是在当前终端里启动的，按：

```text
Ctrl + C
```

看到终端回到命令输入状态，就表示服务已经关闭。

## 重新启动

关闭后再次运行：

```bash
npm start
```

## 端口被占用

如果启动时提示 `5173` 端口被占用，可以换一个端口启动：

```bash
PORT=5174 npm start
```

对应访问地址也要改成：

```text
http://localhost:5174
```

## 功能

- 在线玩家列表
- 玩家邀请与接受/拒绝
- 双人五子棋对局
- AI 模式单人对局
- 胜负判定、平局判定、认输
- 对手离线自动结束对局
